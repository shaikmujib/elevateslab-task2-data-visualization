# elevateslab-task2-data-visualization

## Task 2: Data Visualization and Storytelling
This repository contains Task 2 of the Elevates Lab Data Analyst Internship, focused on creating meaningful visualizations and deriving business insights using Power BI.

---

## Tool Used
- Power BI Desktop

---

## Dataset
- Name: Superstore Sales Dataset
- Format: CSV
- Data includes sales, profit, category, region, product, and date information.

---

## Visualizations Created
1. Sales by Category (Bar Chart)
2. Profit by Category (Bar Chart)
3. Sales by Region (Pie Chart)
4. Monthly Sales Trend (Line Chart)
5. Top 10 Products by Sales (Bar Chart)

---

## Key Insights
- Technology category contributes the highest sales and profit.
- West region is the top-performing region.
- Sales show a steady trend over time.
- A small number of products generate most of the overall revenue.
